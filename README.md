# Pharmacy Analytics Dashboard

This repository contains a sample pharmacy business analytics dashboard built using Excel.

The purpose of this dashboard is to demonstrate how pharmacy operational data can be transformed into insights that support better decision making.

## Dashboard Capabilities

The dashboard provides visibility into:

- Total revenue
- Total units sold
- Total profit
- Average daily sales
- Revenue by medication category
- Profit by category
- Top selling medications
- Daily revenue trends

## Business Insights

With the right data structure, pharmacy owners can easily monitor:

- Fast-moving medications
- Sales performance trends
- Profitability by product category
- Customer demand patterns
